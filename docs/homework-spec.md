# Homework Specification

## Goal

The objective of this assignment is to develop a web-based platform that monitors and visualizes selected financial indicators to assist investment and trading decisions.

In particular, the focus is on indicators related to DAT.co (Digital Asset Treasury companies). These indicators are commonly used in crypto markets to evaluate the valuation and behavior of companies holding Bitcoin or other digital assets.

By building this system, students gain experience in:

- Financial data collection
- Indicator analysis
- Web-based visualization
- Optional AI-assisted insights

## Task Description

Design and implement a website that allows users to observe the time-series behavior of a chosen DAT.co-related indicator.

## Requirements

### 1. Select an Indicator

Choose one indicator related to DAT.co.

Examples include, but are not limited to:

- mNAV (Modified Net Asset Value)
- Premium to NAV
- Discount to NAV

### 2. Data Collection

Identify and obtain the data source for the chosen indicator.

Possible approaches include:

- Using public APIs
- Web scraping from financial or crypto-related websites
- Aggregating data from multiple sources

### 3. Data Visualization

Build a user interface that allows users to:

- Observe the indicator over time at daily frequency

### 4. AI-Generated Summary (Optional Bonus)

Integrate any LLM API, such as:

- OpenAI API
- Gemini API
- Claude API

Potential uses:

- Automatically generate summaries or insights based on the indicator data
- Provide interpretation or trend analysis

### 5. Deployment

Deploy the website to a publicly accessible domain.

Possible platforms include:

- Vercel
- GCP
- AWS

### 6. Report Writing

Submit a report describing the work.

## Report Requirements

The report should include:

### Selected Indicator

- What indicator was chosen?
- Why was it chosen?

### Relationship with Bitcoin (BTC)

- Explain how the indicator is related to BTC
- Provide insights or hypotheses about how it affects or reflects BTC price behavior

### Deployed Website URL

- Provide the link to the live website
